## 📦 Getting Started
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Anurag03singh/portfolio.git
   cd portfolio
   ```
2. **Install dependencies:**
   ```bash
   yarn install
   # or
   npm install
   ```
3. **Run the development server:**
   ```bash
   yarn dev
   # or
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
